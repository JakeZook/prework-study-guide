Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The "html lang="en"" will let the browser know the language used is English.
  • The title will display text on the tab for the webpage.
  • "h1", "h2", "h3', "h4" are different sized headers for strings.
  • "img src" will provide the location of the image and "alt" will provide text for the image.

GIT

  • cd: Move into a folder or directory.
  • open: Opens a file.
  • code.: Opens file in VSCode.
  • git status: Checks what branch we are currently on.
  • git checkout -b "branch-name": Creates a new branch and switches to it.
  • git add -A: Commits all files in the working branch.
